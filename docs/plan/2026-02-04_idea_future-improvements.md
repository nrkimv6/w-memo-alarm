## UX 개선

### 커맨드 팔레트 (Cmnd+K)

**출처**: `2026-01-08_memo-alarm-improvement-plan.md`

마우스 없이 키보드만으로 앱의 모든 기능을 제어할 수 있는 인터페이스.

**기능**:
- 바로가기: `Ctrl+K` (Win) / `Cmd+K` (Mac)
- 메모 검색 (제목, 내용, 태그)
- 새 메모 작성
- 테마 변경 (다크/라이트)
- 보기 모드 변경 (리스트/그리드)
- 휴지통 비우기

**기술 스택**:
- `cmdk` 또는 `svelte-command-palette` 라이브러리

---

### View Transitions 고급 전환

**출처**: `2026-01-08_memo-alarm-improvement-plan.md`

**미구현 효과**:
- 메모 카드 → 상세 모달: 카드가 커지면서 모달로 변신
- 메모 삭제: 카드가 사라지며 빈 공간이 자연스럽게 채워짐
- 폴더 이동: 탭 전환 시 내용이 슬라이딩

## 지능형 기능

### 로컬 AI 스마트 태깅

**출처**: `2026-01-08_memo-alarm-improvement-plan.md`

서버 통신 없이 로컬 로직(RegEx, Heuristics)만으로 똑똑한 제안 제공.

#### 스마트 알림 제안
- **내용 분석**: "내일 오후 2시 미팅" 입력 시
- **동작**: 날짜(내일)와 시간(14:00)을 추출하여 알림 설정 칩(Chip) 표시
- **키워드**: 오늘, 내일, 모레, 아침, 점심, 저녁, 분 후, 시간 후

#### 자동 태그 추천
- **키워드 매칭**:
  - "우유", "계란", "마트" → `#장보기`
  - "미팅", "회의", "제출" → `#업무`
- **학습**: 사용자가 자주 사용하는 태그 패턴을 localStorage에 저장하여 정확도 향상

## 동기화

### 이중 동기화 모드 (Dual Sync)

**출처**: `2026-01-11_memo-alarm-dual-sync-mode.md`

**상태**: 타입 정의만 완료 (20%)

사용자가 선택할 수 있는 두 가지 동기화 방식:
- **간편 모드**: 6자리 코드 (빠른 연결, 보안 약함)
- **안전 모드**: 로그인 (OAuth, 보안 강화, 계정 복구 가능)

**미구현**:
- `unifiedSyncStore` 스토어
- 모드 선택 UI (설정 페이지)
- 코드 ↔ 로그인 전환 로직
- 데이터 마이그레이션 (코드 → 로그인)
- Realtime과 통합

**참고**: `memo-alarm/docs/archive/2026-01-11_memo-alarm-dual-sync-mode.md`
