# Sacred Hours 테마 선택기 확장

> 작성일: 2026-01-07
> 대상 프로젝트: sacred-hours
> 상태: ✅ 완료 (2026-01-08)

---

## 개요

현재 다크/라이트 모드 토글만 있는 sacred-hours에 가톨릭 전통 색상 테마를 추가하여 사용자가 선호하는 분위기를 선택할 수 있게 합니다.

---

## 테마 제안

### 1. 가톨릭 전통 색상 조합

| 테마명 | 주요 색상 | 설명 |
|--------|----------|------|
| **버건디 골드** | 버건디 #722F37 + 금색 #D4AF37 | 클래식하고 장엄한 느낌, 현재 기본 테마 |
| **마리안 블루** | 남색 #1E3A5F + 금색 #C9A227 | 성모 마리아 상징색, 차분하고 경건함 |
| **아이보리 골드** | 아이보리 #FFFFF0 + 금색 #B8860B | 밝고 따뜻한 느낌, 축일 분위기 |

### 2. 남색 계열 (Marian Blue)

| 테마명 | 배경 | 악센트 | 텍스트 |
|--------|------|--------|--------|
| **남색 라이트** | #F0F4F8 | #1E3A5F | #1A1A2E |
| **남색 다크** | #0D1B2A | #4A90D9 | #E0E7EE |

### 3. 전례력 색상 (신부님 제의)

| 테마명 | 시기 | 주요 색상 | 배경/악센트 |
|--------|------|----------|-------------|
| **대림/사순** | 대림절, 사순절 | 보라 #6B3FA0 | 연보라 배경 |
| **연중 시기** | 일반 주일 | 녹색 #2E7D32 | 연녹색 배경 |
| **부활/성탄** | 축일 | 흰색/금색 | 밝은 배경 + 금색 악센트 |
| **성령/순교** | 성령강림, 순교자 축일 | 빨강 #B71C1C | 연빨강 배경 |
| **장미** | 대림3주, 사순4주 | 장미색 #E91E63 | 연분홍 배경 |

## 기술적 고려사항

- **CSS 변수 구조**: 기존 HSL 변수 시스템 유지
- **저장소**: localStorage의 `theme` 키 확장 (예: `theme-id`)
- **다크모드 조합**: 각 테마에 라이트/다크 변형 제공
- **접근성**: 색상 대비 WCAG AA 기준 충족
