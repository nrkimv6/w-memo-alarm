# 홈/전체메모 탭 분리 계획서

> **작성일**: 2026-01-23
> **타입**: 기능 개선
> **목표**: 홈 탭과 전체 메모 탭 분리로 UX 개선

---

## 요구사항

1. **홈 탭 검색창**: 기본 숨김 → 검색 아이콘 클릭 시 노출
2. **홈 탭 메모 제한**: 핀, 즐겨찾기, 알림예정, 최신메모만 표시
3. **전체 메모 탭**: 하단 네비게이션에 새 탭 추가

---

## 현재 구조

```
BottomNav
├── 홈 (/) - 모든 메모 + 검색 + 필터
└── 설정 (/settings)
```

## 변경 후 구조

```
BottomNav
├── 홈 (/) - 대시보드 스타일
│   ├── 검색 아이콘 (클릭 시 검색창 표시)
│   ├── 📌 고정된 메모 (최대 5개)
│   ├── ⭐ 즐겨찾기 (최대 5개)
│   ├── 🔔 알림 예정 (오늘/내일)
│   └── 🕐 최신 메모 (최대 5개)
├── 전체 메모 (/memos) - 기존 홈 기능 이동
│   ├── 검색창 (항상 표시)
│   ├── 필터/정렬
│   └── 모든 메모 목록
└── 설정 (/settings)
```

### Phase 2: 전체 메모 페이지 생성

**파일**: `src/routes/memos/+page.svelte`

- 기존 홈(+page.svelte)의 전체 메모 기능 이동

- FilterTabs, FolderTabs, TagFilter 유지
- 모든 메모 목록 표시

### Phase 3: 홈 탭 리디자인

**파일**: `src/routes/+page.svelte`

#### 3.1 검색 토글
```svelte
let showSearch = $state(false);

<!-- 검색 아이콘 -->
<Button onclick={() => showSearch = !showSearch}>
    <Search class="w-4 h-4" />
</Button>

<!-- 검색창 (조건부 표시) -->
{#if showSearch}
    <SearchBar />
{/if}
```

#### 3.2 대시보드 섹션
```svelte
<!-- 고정된 메모 -->
<section>
    <h2>📌 고정된 메모</h2>
    {#each pinnedMemos.slice(0, 5) as memo}
        <MemoCard {memo} compact />
    {/each}
</section>

<!-- 즐겨찾기 -->
<section>
    <h2>⭐ 즐겨찾기</h2>
    ...
</section>

<!-- 알림 예정 -->
<section>
    <h2>🔔 오늘/내일 알림</h2>
    ...
</section>

<!-- 최신 메모 -->
<section>
    <h2>🕐 최신 메모</h2>
    ...
</section>
```
