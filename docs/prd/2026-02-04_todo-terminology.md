# 할일(Todo) 용어 정리

> **작성일**: 2026-02-04
> **관련 문서**: [할일 메모 타입 PRD](./2026-02-04_todo-note-prd.md)

---

## 알림 3단계

| 한국어 | 영어 | 코드명 | 설명 |
|--------|------|--------|------|
| **상기** | Remind | `remind` | 매일 설정 시각(예: 오전 7시)에 발송하는 가벼운 요약 알림. 기한 없는 할일도 포함. 일반 푸시 노티피케이션. |
| **알람** | Alert | `alert` | 사용자가 "이때 하겠다"고 마음먹은 시점의 강한 알림. 발등에 불. 진동/소리 + 화면 전체(Native) 또는 고정 노티피케이션. |
| **기한초과** | Overdue | `overdue` | 기한 시각이 지난 상태. 별도 알림을 발송하지 않으며, 앱 내에서 빨간 배지 + 목록 상단 고정으로 표시. 연산된 상태값. |

### 상기 vs 알람 vs 기한

```
상기 (Remind)  → "잊지 마세요" (정보 전달)
알람 (Alert)   → "지금 하세요!" (행동 촉구)
기한 (Deadline) → 기준 시각. 경과하면 overdue (상태 표시)
```

## 할일 상태

| 한국어 | 영어 | 코드명 | 설명 |
|--------|------|--------|------|
| **미완료** | Pending | `pending` | 아직 처리하지 않은 상태 |
| **완료** | Completed | `completed` | 사용자가 완료 체크한 상태 |
| **건너뜀** | Skipped | `skipped` | 이번 일정을 수행하지 않고 넘긴 상태. 반복 할일에서 다음 인스턴스 생성됨. |
| **미룸** | Postponed | `postponed` | 일정을 뒤로 미룬 상태. 같은 인스턴스의 `scheduledDate`만 변경됨. |

---

## 할일 액션

| 한국어 | 영어 | 코드명 | 설명 |
|--------|------|--------|------|
| **미루기** | Postpone | `postpone` | 현재 인스턴스의 예정일을 뒤로 변경. 2회차부터 에스컬레이션 경고 + 횟수 제한 입력 가능. |
| **건너뛰기** | Skip | `skip` | 현재 인스턴스를 수행하지 않고 종료. 반복 할일이면 다음 인스턴스가 자동 생성됨. |
| **에스컬레이션** | Escalation | `escalation` | 미루기 2회차부터 발생하는 경고. "정말 미루시겠습니까?" + 최대 횟수 제한 설정 토글. |

## 반복 / 인스턴스

| 한국어 | 영어 | 코드명 | 설명 |
|--------|------|--------|------|
| **반복 할일** | Recurring Todo | `recurrence` | 주기적으로 반복되는 할일. 패턴(daily/weekly/monthly/custom) 보유. |
| **인스턴스** | Instance | `todoInstance` | 반복 할일의 개별 발생건. DB에는 과거 기록 + 현재 활성 1건만 저장. 미래 일정은 연산으로 표시. |
| **직전 생성** | Just-in-Time (JIT) | - | 인스턴스 생성 전략. 현재 인스턴스 완료/건너뛰기 시 다음 인스턴스 1개만 생성. |

## 설정 용어

| 한국어 | 영어 | 코드명 | 설명 |
|--------|------|--------|------|
| **상기시간** | Remind Time | `remind.time` | 매일 상기 알림을 받을 시각. 앱 공통설정으로 on/off + 시각 지정. |
| **자동알람** | Auto Alert | `autoAlert` | 기한 N시간 전에 자동으로 알람을 발송하는 기능. 기한 없는 할일에는 적용 불가. |
| **공통설정** | Global Settings | `todoDefaults` | 앱 전체에 적용되는 할일 기본값. todo별 개별설정으로 덮어쓸 수 있음. |

---

## 데이터 조회

| 한국어 | 영어 | 설명 |
|--------|------|------|
| **실제 데이터** | Real Data | DB에 저장된 인스턴스 (과거 완료/건너뜀 + 현재 활성 pending). |
| **계산된 데이터** | Computed Data | 반복 패턴으로부터 메모리에서 계산한 미래 예정일. DB에 저장하지 않음. |
| **할일 모아보기** | Todo Overview | 일정 기준으로 할일을 통합 조회하는 뷰. 실제 데이터 + 계산된 데이터 합산 표시. |

---

## 기존 앱 용어와의 관계

| 기존 용어 | 할일에서의 대응 | 비고 |
|-----------|----------------|------|
| Reminder (기존 알람) | 특정 시간에 알려줌 | 완료 개념 없음, 단순 알림 |
| Todo 상기 (Remind) | 매일 할일 요약 알림 | 할일 전용, 기한 없는 할일도 포함 |
| Todo 알람 (Alert) | 행동 촉구 알림 | 발등의 불, 강한 알림 |
| Bookmark URL 자동열기 | - | 할일과는 무관 |

## 용어 변경 이력

| 변경 전 | 변경 후 | 사유 |
|---------|---------|------|
| 알림 (Alert) | **알람** (Alert) | "알림"은 일반 notification과 혼동됨. "알람"이 "지금 해야 한다"는 긴박감을 더 잘 전달. |
