## ⚠️ 주의사항

이 문서의 기능들은 **필수 기능이 아닌 선택적 고급 기능**입니다.
핵심 기능(Phase 1-12)은 이미 모두 완료되었으며, 아래 기능들은 필요 시 검토합니다.

## Phase 18: 자동화 & 연동

- [ ] **IFTTT / Zapier 연동**

## Phase 23: ML Kit & On-Device AI

**필요 플러그인**: `@capacitor/camera`, `@nicholasrobinson/capacitor-mlkit-*`, `@capacitor-mlkit/barcode-scanning`

- [ ] **사진 첨부 & 갤러리**
  - 메모에 다중 이미지 첨부
  - 카메라 직접 촬영
  - 갤러리에서 선택
  - 이미지 미리보기 그리드

- [ ] **ML Kit - 텍스트 인식 (OCR)**
  - 사진에서 텍스트 추출
  - 스크린샷 → 메모 변환
  - 명함 스캔 → 연락처 메모
  - 손글씨 인식

- [ ] **ML Kit - 바코드/QR 스캔**
  - QR 코드 → URL 북마크 저장
  - 바코드 → 상품 정보 메모
  - 카메라 실시간 스캔

- [ ] **ML Kit - 이미지 라벨링**
  - 사진 자동 태그 생성
  - 음식/장소/물건 인식
  - 스마트 앨범 분류

## Phase 24: Galaxy S25 로컬 LLM (Samsung AI)

**필요 플러그인**: Samsung AI SDK (커스텀 플러그인)

- [ ] **로컬 LLM 메모 요약**
  - 긴 메모 → 3줄 요약
  - 하이라이트 추출
  - 키워드 자동 추출

- [ ] **로컬 LLM 태그 추천**
  - 메모 내용 분석 → 태그 제안
  - 기존 태그와 매칭
  - 새 태그 생성 제안

- [ ] **로컬 LLM 관련 메모 찾기**
  - 현재 메모와 유사한 메모 추천
  - 의미 기반 검색 (키워드가 아닌)
  - 연관 키워드 확장

- [ ] **로컬 LLM 글쓰기 도우미**
  - 문장 자동 완성
  - 문법/맞춤법 교정
  - 다듬기 제안

- [ ] **로컬 LLM 할 일 추출**
  - 메모에서 액션 아이템 추출
  - 자동 체크리스트 생성
  - 마감일 파싱
